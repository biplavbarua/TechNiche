# Tech Stack

## Frontend
- **Framework**: Next.js 15 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4, PostCSS
- **Animations/Graphics**: Framer Motion, Three.js
- **Icons**: Lucide React
- **Testing**: Playwright

## Backend
- **Framework**: FastAPI
- **Server**: Uvicorn
- **Validation**: Pydantic
- **Vector Database**: Pinecone
- **AI/LLM**: OpenRouter API, Google GenAI
- **Web Scraping**: BeautifulSoup4
- **Testing**: Pytest

## Environment & Tooling
- **Language**: Python 3.10+, JavaScript/TypeScript
- **Environment Management**: `.env` files
- **Package Managers**: npm (Frontend), pip (Backend)
